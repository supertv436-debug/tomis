Кафедральный AI-ассистент v4.3

Автоматизированная система для создания учебной документации кафедры «Технологическое оборудование, машиностроение и стандартизация».

Возможности:

  • Автоматическая генерация календарных планов
  • Создание экзаменационных билетов из СРС
  • Интеллектуальный анализ силлабусов
  • Экспорт расписаний из Excel
  • Smart Fill - заполнение шаблонов DOCX данными

Технические характеристики:

  • Модель AI: Llama 3.3 70B (Groq API)
  • Форматы: PDF, DOCX, XLSX
  • Макс. размер файла: 10 МБ
  • Шаблоны: плейсхолдеры {{ПОЛЕ}} для автозаполнения
