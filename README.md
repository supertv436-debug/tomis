## 🚀 Быстрый старт

### В браузере (рекомендуется):
1. Откройте `index.html` в браузере
2. Настройте API ключ Groq
3. Загрузите документы
4. Начните работу

### На GitHub Pages:
1. Залейте код в репозиторий
2. Активируйте GitHub Pages
3. Сайт будет доступен по адресу `https://username.github.io/repository`

## 🔧 Требования

### Для браузера:
- Современный браузер (Chrome, Firefox, Safari, Edge)
- Интернет-соединение для API запросов
- JavaScript включен

### Для API:
- Ключ Groq API (начинается с `gsk_`)
- Поддерживаемые модели: `llama-3.3-70b-versatile`

## 📋 Инструкция по использованию

### 1. Настройка API ключа
- Перейдите в "Настройки"
- Введите API ключ Groq
- Нажмите "Сохранить"

### 2. Загрузка документов
- Перейдите в "База данных"
- Перетащите PDF/DOCX файлы
- Дождитесь обработки

### 3. Работа с AI-ассистентом
- Перейдите в "Нейросеть"
- Задайте вопросы по загруженным документам
- Получайте точные ответы

### 4. Генерация документов
- Перейдите на "Панель управления"
- Нажмите "План" или "Экзамен"
- Результат скопируется в буфер обмена

### 5. Экспорт расписаний
- Перейдите в "Экспорт данных"
- Загрузите Excel файл с расписанием
- Нажмите "Обработать расписание"

## 🎨 Особенности интерфейса

### Deep Black Interface:
- **Черный фон** (#000000) - максимальная контрастность
- **Серые слои** (#0A0A0A) - для панелей
- **Акцентный синий** (#2F6BFF) - активные элементы
- **Минималистичный дизайн** - профессиональный вид

### Адаптивность:
- Работает на всех устройствах
- Оптимизирован для мобильных
- Поддержка touch-управления

## 📊 Статистика и аналитика

Система автоматически отслеживает:
- Количество загруженных документов
- Обработанных запросов к AI
- Сгенерированных документов
- Экспортированных данных

## 🔒 Безопасность

- **Локальное хранение** - данные не покидают браузер
- **API ключ** хранится только в localStorage
- **Ни одного сервера** - полностью клиентское приложение
- **HTTPS соединения** для API запросов

## 🌐 Технологии

### Frontend:
- **HTML5** - современная разметка
- **Tailwind CSS** - стилизация
- **JavaScript ES6+** - логика приложения
- **Font Awesome** - иконки

### Библиотеки:
- **PDF.js** - обработка PDF файлов
- **Mammoth.js** - обработка DOCX файлов
- **Groq API** - AI функциональность

### Деплой:
- **GitHub Pages** - статический хостинг

## 📱 Поддерживаемые форматы

- **TXT** - экспорт контекста

## Использование

1. **Установите API ключ**: Введите ваш Groq API ключ (начинается с `gsk_`)
2. **Загрузите силлабус**: Перетащите PDF/DOCX файл или выберите через кнопку
3. **Задавайте вопросы**: Используйте чат-интерфейс для диалога с ИИ
4. **Генерируйте документы**: Создавайте календарные планы и экзаменационные билеты

- Python 3.7+
- Groq API ключ
